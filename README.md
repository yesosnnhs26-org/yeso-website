# YES-O Website

A simple static website for the Youth for Environment in Schools Organization.

## Files

- `index.html` — website structure and content
- `style.css` — design, layout, colors, responsiveness
- `script.js` — menu and scroll reveal animations
- `assets/` — place your logo and photos here

## How to use with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save, then wait for GitHub Pages to publish the site.

## Customization

Replace the logo placeholder in `index.html` with an actual image, and edit the sections to match your official YES-O content.
